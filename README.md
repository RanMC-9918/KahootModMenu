# Kahoot Mod Menu
A kahoot mod menu that has options like AI answer finder and controllable bots

Made with [kahoot.js-latest](https://www.npmjs.com/package/kahoot.js-latest)

**!IMPORTANT!: WILL NOT WORK WITHOUT API URL**
